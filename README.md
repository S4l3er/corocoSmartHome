# Smart Home Automation Platform

A production-grade Smart Home Automation Platform designed and built from scratch covering hardware, backend infrastructure, automation workflows, and real-time communication.

## Features

* Real-time device communication using MQTT
* Workflow-driven automation engine
* Visual IF / THEN automation builder
* Role-based access control
* Dynamic space and device management
* Environmental monitoring
* Smart switching and automation

## Technology Stack

* Node.js / Express
* React / Redux
* ESP32
* MQTT
* MongoDB
* REST APIs

## Architecture

The platform uses event-driven communication and distributed device orchestration to provide scalable automation capabilities while preserving real-time responsiveness.
